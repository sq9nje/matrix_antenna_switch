OTRSP (SO2R Protocol)

Listens on TCP port 12060. Used by N1MM+, WriteLog, and Win-Test for SO2R control. Restart required.
When enabled, the RS-485 port uses OTRSP protocol (9600 baud) instead of the native serial protocol
